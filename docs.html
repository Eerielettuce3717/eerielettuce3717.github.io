Online IDE Documentation

This guide explains how to use Nexus Online IDE — a browser-based online IDE and web code editor for running JavaScript, Python, HTML/CSS, and other languages without local installation.

Open the editor

Getting started

Visit https://eerielettuce3717.github.io/ and the editor loads in your tab. Create or switch files from the tab bar. Select a language per file — Python, JavaScript, HTML, CSS, C++, Java, and others are supported. Write code in the Monaco editor, then press Run or Ctrl+Enter to execute the active file.

Running code & console output

JavaScript runs in the browser engine. Python runs via Pyodide (WebAssembly CPython). Output appears in the Terminal panel. Errors show line numbers when available. For Python input(), press Run first — the terminal prompts for each line as your program executes.

Web preview

When editing HTML, CSS, or JavaScript, open the Preview tab. The panel combines your project files into a live document so you can see layout and behavior without leaving the IDE.

Terminal & packages

The terminal supports REPL-style Python lines, pip install via Pyodide micropip (pip install numpy), and optional connection to a local shell server for full system commands on your machine.

Save & share code

Sign in with Google or GitHub to save projects to GitHub. Use the GitHub tab to push files to a repository. Export by copying from the editor or saving through GitHub OAuth with repo scope.

Keyboard shortcuts

  • Ctrl+Enter — Run active file
  • Ctrl+F / Ctrl+H — Find and replace
  • Ctrl+L — Clear terminal (when focused)
